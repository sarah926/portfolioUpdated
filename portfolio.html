Product Analysis:

See on Github

Problem Identified: Showcasing my coding skills was incomplete on a one page resume.

KPI: Reach 100 pageviews in 2 months after deployment.

Goals:

  • Develop a live web page as an extension of my resume.
  • Create a user friendly interface to easily navigate through the website with no latency.
  • Include animations and personal features to help others get to know me and my skills.
  • Practice and further develop my HTML, CSS and Javascript fluency.
Role: Backend and frontend developer, UX designer. Technologies Utilized: HTML, CSS, Javascript, Figma. User Journey:
  1. The user clicks on the link from my LinkedIn or resume.
  2. The user is taken to a homepage with a quick summary of me.
  3. The user can easily find the projects section and click on one.
  4. The user is taken to the specific project page, where they can visually see the product and read more information, including a brief synopsis and a more in depth product analysis.
  5. The user can navigate at any time to the home page, about page, or my resume.
  6. The user can click on the about page from the nav bar.
  7. The user is taken to the about page, where they can get more information about me.
